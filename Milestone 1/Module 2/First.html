CSS: Cascading Style Sheets

Types of CSS:

1. Inline css ---- no reusability

2. Internal or Embedded

3. External css


Selectors:

  • Id ----- single element (#id_name)
  • Class ----- multiple elements (.class_name)
  • Global ----- asterisk (*)
  • Grouping ----- style on multiple selector at once (__, __, ...)
  • Attribute ----- follow attribute type (__[type='attribute_name'])
