CSS Positions

  1. Static
  2. Relative
    keep it's own position and will move based on this.
  3. Absolute
    will lose it's own position and will move based on the root (the closest relative parent)
  4. Sticky
    will lose it's own position.
    have to define the position and only stay in immediate parent div
  5. Fixed
    will lose it's own position.
    have to define the position and but stay in every div. actually it consider the root as parent (viewport).
